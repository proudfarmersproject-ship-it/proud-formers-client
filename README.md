## Local Set Up

    1. Clone the Repo to user Local repository
    2. Open the command prompt, jum to the project path. ex "c/projects/proud-formers-client"
    3. install dependencies using `npm install`
    4. run the project Using `npm run dev`

    Note: To visualize the data please use second terminal for run data
    'npm run mock"  ==> It will run the db.json
